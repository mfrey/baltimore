baltimore
=========

Installation from git
---------------------
1. Clone the repository:

		git clone git://github.com/mfrey/flow.git

2. Please install the required (python) dependencies

		networkx
		matplotlib
		nose	

3. Run the program

		python2.7 main.py

4. Tests are executed using nose. From the root of the project call

		nosetests
